클래스
-----
클래스 생성
```java
public class Monster(){
  int HP;
  String Name;

  public ReName(String str){
    Name = str;
  }

  public void Damaged(int k){
    HP-=k;
  }

}

```
클래스 호출
``` java
public class Zombie {

  public static void main(String[] args) {
  
    Monster mob = new Monster();
    mob.hp = 100;
    mob.ReName("A1");
    mob.Damaged(25);

  }
}

```
