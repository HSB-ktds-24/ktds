메소드
-----
* 하나의 기능을 하는 코드의 묶음 단위
* 동일한 기능의 코드가 여러 번 중보괴어 있을 경우 메소드를 사용하면 편리함

* **구성**
```java
// 리턴이 없는 메소드 = 리턴타입: void
public  static  void  method1 (int x ){
//접근제한자  리턴타입 메소드명 매개변수
}
//리턴이 있는 메소드
punlic static int method2 (int x){
return x*x;
// 리턴과 함게 메소드 종료
}

public static void main(String[] args) {
  method1(4);
  //메소드명 (매개변수)
  int k = method2(4)
 
```
